# test-fanpl

基于 Lovable + GitHub + Cursor 协同开发的全栈应用项目。

## 技术栈

- **前端**: React + TypeScript + Tailwind CSS
- **构建工具**: Vite
- **后端/数据库**: Supabase
- **AI 开发工具**: Lovable / Cursor

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 项目结构

详见 [目录结构说明](./docs/directory-guide.md)

## 协同工作流

1. **Lovable** — AI 快速生成原型和迭代
2. **GitHub** — 版本控制和团队协作
3. **Cursor** — 深度开发和代码优化
